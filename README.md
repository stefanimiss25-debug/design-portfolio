# Stefania Portfolio

Персональный сайт-портфолио для AI-креатора и дизайнера Стефании.

Сайт выполнен в тёмной атмосферной стилистике с акцентом на:
- глубину и воздух в композиции,
- премиальные анимации,
- визуальную подачу кейсов.

## Технологии

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla)`

## Структура

- `index.html` — основная разметка сайта
- `styles.css` — стили, визуальная система, анимации
- `script.js` — интерактив, reveal/hover/lightbox/параллакс
- `assets/` — изображения для кейсов

## Разделы сайта

1. Hero
2. Обо мне
3. Подход
4. Портфолио (сайт, AI-креатив, афиши, кейсы айдентики, логотипы)
5. Услуги
6. Контакты

## Локальный запуск

Открой терминал в папке проекта и выполни:

```bash
python3 -m http.server 5500
```

После запуска открой:

`http://localhost:5500`

Остановка сервера:

`Ctrl + C`
